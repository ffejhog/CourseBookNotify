Welcome to GitHub Pages.

As most classes at UT Dallas do not allow wait listing, this app was developed to allow you to monitor when a class is no longer 100% full though the use of notifications and use of the app interface. The interface displays the exact percentage that represents the number of students enrolled in the class. Note this app is in beta and still in development.

This app is very basic and was to teach me android development. Maybe it will help someone get a class they want/need:D

The code is very messy. I will clean it up and comment it better when I have time.
